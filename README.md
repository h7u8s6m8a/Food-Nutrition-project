# Food-Nutrition-project
Developed an end-to-end Food Nutrition Analysis project using pandas for data cleaning and preprocessing, and power BI for interactive dashboard visualization. 

 ### Project Overview:

The Food Nutrition Analysis Dashboard is an end-to-end data analytics project designed to analyze and visualize nutritional information of various food items.
This project demonstrates practical skills in data cleaning, transformation, modeling, and interactive dashboard development using Python (Pandas) and Power BI.
The objective is to convert raw nutritional data into meaningful insights through structured preprocessing and visually compelling reports.
